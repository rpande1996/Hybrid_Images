## Overview

This code creeates an hybrid image of an input image pair

## Softwares

* Recommended IDE: PyCharm 2021.2

## Libraries

* Numpy 1.21.2
* OpenCV 4.6.0
* Matplotlib 3.5.0

## Programming Languages

* Python 3.8.12
